Go のローカルインストール

Mac

インストーラー経由でインストール（推奨）

こちら からダウンロードして実行

Homebrew を使用してインストール（バージョンが古い場合あり）

  1. brew install go
  2. go version でバージョンが表示されれば OK

Windows

こちら からインストーラーをダウンロードして実行

Linux

  1. rm -rf /usr/local/go && tar -C /usr/local -xzf go1.19.2.linux-amd64.tar.gz
  2. vi ~/.bashrc で export PATH=$PATH:/usr/local/go/bin を追記
  3. source ~/.bashrc
  4. go version でバージョンが表示されれば OK
